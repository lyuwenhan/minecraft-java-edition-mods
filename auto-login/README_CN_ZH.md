# 自动登录 

一个客户端 Fabric 模组，可在加入需要认证的 Minecraft 服务器（如 EasyAuth、AuthMe）时自动登录，密码采用本地加密存储。

本模组完全在客户端运行，不会修改任何服务器行为。

---

## 功能特性

- 加入服务器后自动发送 `/login <密码>` 命令
- 密码本地加密存储（AES + PBKDF2）
- 无需服务器端插件或模组
- 仅客户端，适用于多人服务器
- 支持通过命令手动触发，用于测试或备用

---

## 命令

### 设置密码（首次使用需要）

```
/autologin set <密码>
```

- 对密码进行加密并本地保存
- 自动启用自动登录功能

### 立即触发自动登录

```
/autologin login
```

- 立即执行登录尝试
- 适用于测试或手动重试

### 启用 / 禁用

```
/autologin on
/autologin off
/autologin toggle
```

### 清除已保存的密码

```
/autologin clear
```

- 删除存储的凭据
- 禁用自动登录

---

## 工作原理

1. 首次设置时，密码被加密并本地保存
2. 当你加入服务器时：
   - 模组等待客户端完全初始化
   - 然后直接向服务器发送登录命令
3. 密码绝不会发送到其他地方，也不会被记录

本模组不会拦截数据包、修改 UI 或 hook 服务器认证逻辑。

---
# 安全说明

- 密码仅存储在你的本地机器上
- 加密使用：
  - PBKDF2（密钥派生）
  - AES-GCM（认证加密）
- 明文密码不会写入磁盘
- 请勿复用重要的真实世界密码
