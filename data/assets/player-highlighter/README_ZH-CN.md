# Player Highlighter (玩家高亮器)

一个轻量级的**客户端 Fabric 模组**，可在按住指定按键或启用切换模式时，为玩家显示发光轮廓高亮，且不依赖任何服务端机制。

---

## 概述

玩家高亮器 可通过在**按住高亮按键时（默认：TAB）**，或**启用高亮切换模式时**，为其他玩家显示发光轮廓，从而提升玩家可见性。  
该效果是纯视觉性的，完全由客户端处理，并且可根据用户偏好设为短暂触发或持续生效。

当既没有按住高亮按键，也没有启用切换模式时，所有轮廓都会立即消失。

---

## 核心特性

- 在**按住已配置按键**时，或**启用保持模式**时，显示发光轮廓
- 当高亮激活时，为视野中的其他玩家显示屏幕 HUD 图标指示
- 支持可选的切换模式以实现持续高亮（默认按键：**I**）
- 未激活时不会造成屏幕杂乱
- 纯客户端，仅视觉效果
- 不会修改玩家的真实状态或效果
- 无命令、无数据包、无玩法改动
- 可安全用于多人服务器
- 轻量化，性能影响可忽略不计

---

## 视觉表现

- 其他玩家会以**发光**效果的轮廓被渲染出来
- 本地玩家自己不会被高亮
- 轮廓显示由以下方式控制：
  - 按住高亮按键（默认：**TAB**），或
  - 通过切换按键启用保持模式（默认：**I**）
- 激活时，屏幕上会出现小型 HUD 图标，用于指示视野中玩家的位置
- 轮廓颜色遵循原版规则（默认颜色或队伍颜色，若适用）
- 不会影响生物或非玩家实体

---

## 多人游戏安全性

- 仅客户端
- 不与服务器交互
- 不会拦截或发送任何数据包
- 不存在不同步风险
- 兼容原版服务器和模组服务器

---

## 支持版本

- Fabric Loader 0.18+
- Fabric API
- Java 21
